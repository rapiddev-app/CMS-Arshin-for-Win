## 🚀 Процесс сборки (Вариант 2)

### Шаг 1: Подготовка Embedded Python

```bash
python prepare_embedded_python.py
```

**Что происходит:**
1. Скачивается Python 3.11.9 Embedded (~30 MB)
2. Распаковывается в `python_embedded/`
3. Настраивается `python311._pth`
4. Устанавливается pip
5. Устанавливаются зависимости
6. Проверяется работоспособность

**Время:** 3-5 минут
**Результат:** `python_embedded/` (~120 MB)

---

### Шаг 2: Сборка release

```bash
python build.py
```

**Что происходит:**
1. Очищаются старые сборки
2. Собирается `ScriptManager_Setup.exe`
3. Копируется `python_embedded/` в `release/`
4. Копируются все файлы (src, config, doc)
5. Создаётся `УСТАНОВКА.txt`

**Время:** 2-3 минуты
**Результат:** `release/` (~150 MB)

---

### Шаг 3: Распространение

```bash
# ZIP архив
cd release
7z a ScriptManager_v1.0.0_Standalone.zip *

# Или скопировать на флешку/сетевой диск
```

## 🖥️ Процесс установки (на целевом компьютере)

### Требования:
- ✅ Windows 10/11 (64-bit)
- ✅ 200 MB свободного места
- ✅ Права администратора
- ❌ Python НЕ требуется!

### Установка:

1. Распаковать `release/`
2. Запустить `ScriptManager_Setup.exe` (от администратора)
3. Выбрать путь установки
4. Нажать "Установить"
5. Дождаться завершения (2-3 минуты)
6. Готово!

### Что устанавливается:

```
C:\Program Files\ScriptManager\

│   ├── python.exe
│   ├── Lib/site-packages/
│   └── ...

```

## 🎓 Технические детали

### Настройка python311._pth

**Изменение:**
```diff
  python311.zip
  .

  # Uncomment to run site.main() automatically
- #import site
+ import site
+
+ # Custom paths
+ Lib\site-packages
```

**Зачем:** Включить поддержку site-packages для pip-пакетов

---

### Логика определения типа Python

**В installer.py:**
```python
embedded_python = self.install_dir / "python_embedded" / "python.exe"

if embedded_python.exists():
    # Используем встроенный Python
    f.write('python_embedded\\python.exe src\\gui.py\n')
else:
    # Используем системный Python
    f.write('python src\\gui.py\n')
```

**Преимущество:** Один инсталлятор работает для обоих вариантов!
