# Идея продукта: Менеджер автоматизации скриптов для Windows

## Цель
Создать простое и понятное мини-приложение для Windows, которое автоматизирует управление скриптами по сбору и отправке данных. Приложение должно быть доступно для пользователей с минимальными техническими навыками.

## Целевая аудитория
Обычные пользователи, которые не имеют глубоких технических знаний и взаимодействуют с компьютером на базовом уровне.

## Основные требования

### 1. Управление скриптами
- Единая оболочка для управления всеми скриптами, расположенными в корневой папке по подпапкам
- Централизованное управление запуском всех скриптов из одного интерфейса
- Возможность включения/отключения отдельного скрипта без удаления его настроек

### 2. Настройка расписания
- Выбор времени запуска для каждого скрипта
- Режим запуска:
  - Ежедневно
  - Только в рабочие дни (согласно производственному календарю РФ)

### 3. Управление файлами
- Возможность указать папку с исходным файлом для обработки
- Возможность указать папку для размещения результирующего файла

### 4. Настройки отправки (защищенный раздел)
Отдельное окно с защитой паролем для настройки параметров электронной почты:
- Адрес получателя
- Адрес отправителя
- Тема письма
- Тело письма
- Возможность отправки тестового письма с файлом на указанную почту для проверки работоспособности

### 5. Системные требования
- Запуск через ярлык
- Работа на любом компьютере с ОС Windows 10 и выше
- Портативность - возможность работы без специальной установки

## Ожидаемые преимущества
- Простота использования для неопытных пользователей
- Снижение вероятности ошибок при настройке и запуске скриптов
- Автоматизация рутинных задач по расписанию
- Безопасное хранение настроек отправки писем
