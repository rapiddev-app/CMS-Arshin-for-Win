# 🚀 Быстрый старт ScriptManager

> Пошаговая инструкция по первому запуску и настройке

## Шаг 1: Проверка требований

**Убедитесь, что установлено:**
- ✅ Windows 10/11
- ✅ Python 3.11.9 или выше
- ✅ Excel файлы находятся в OneDrive
- ✅ Доступ к интернету (для SMTP и Telegram)

**Проверка Python:**
```bash
python --version
# Должно быть: Python 3.11.9 или выше
```

## Шаг 2: Установка зависимостей

1. Откройте командную строку в папке проекта:
```bash
cd "e:\All scripts\CMS Arshin for Win"
```

2. Создайте виртуальное окружение (опционально):
```bash
python -m venv venv
venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

**Ожидаемый вывод:**
```
Successfully installed customtkinter-5.2.0 pandas-2.1.3 openpyxl-3.1.2 pywin32-306 requests-2.31.0
```

## Шаг 3: Настройка email

### Gmail (для большинства скриптов)

1. Откройте [config/email.json](config/email.json)

2. Замените данные Gmail:
```json
{
  "gmail": {
    "smtp_server": "smtp.gmail.com",
    "smtp_port": 465,
    "username": "ваш-email@gmail.com",
    "password": "ваш-app-password"
  }
}
```

**Как получить App Password для Gmail:**
1. Перейдите на https://myaccount.google.com/security
2. Включите двухфакторную аутентификацию
3. Перейдите в "App passwords"
4. Создайте новый пароль для приложения "Mail"
5. Скопируйте 16-значный код
6. Вставьте в `password` (без пробелов)

### Yandex (для скрипта Kvadra)

```json
{
  "yandex": {
    "smtp_server": "smtp.yandex.ru",
    "smtp_port": 465,
    "username": "ваш-email@yandex.ru",
    "password": "ваш-пароль"
  }
}
```

## Шаг 4: Настройка Telegram

### Создание бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)

2. Отправьте команду:
```
/newbot
```

3. Укажите название бота (например: `ScriptManager Notifier`)

4. Укажите username бота (например: `scriptmanager_bot`)

5. Скопируйте токен бота (выглядит как: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Получение Chat ID

**Личный чат:**
1. Откройте бота в Telegram
2. Отправьте любое сообщение
3. Откройте в браузере:
```
https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
```
4. Найдите `"chat":{"id":312809937}` - это ваш chat_id

**Групповой чат:**
1. Добавьте бота в группу
2. Отправьте сообщение в группу
3. Откройте в браузере тот же URL
4. Найдите `"chat":{"id":-1002458835967}` - это group_chat_id

### Заполнение config/telegram.json

```json
{
  "bot_token": "YOUR_BOT_TOKEN_HERE",
  "chat_id": "312809937",
  "group_chat_id": "-1002458835967"
}
```

## ❓ Частые проблемы

### Проблема: "Файл не найден"

**Решение:**
1. Откройте `config/scripts.json`
2. Проверьте путь к `input_file`
3. Убедитесь, что файл существует
4. Используйте прямые слеши `/` вместо `\`

### Проблема: "Ошибка при отправке email"

**Решение:**
1. Проверьте `config/email.json`
2. Для Gmail: используйте App Password
3. Для Yandex: включите доступ для почтовых клиентов
4. Проверьте интернет-соединение

### Проблема: "Telegram уведомление не приходит"

**Решение:**
1. Проверьте `config/telegram.json`
2. Проверьте токен бота
3. Проверьте chat_id (откройте getUpdates URL)
4. Убедитесь, что бот добавлен в группу

### Проблема: "Задача не создаётся в Task Scheduler"

**Решение:**
1. Запустите GUI от имени администратора
2. Проверьте права доступа
3. Проверьте логи `logs/app.log`

### Проблема: "Дубль файла реестра"

**Решение:**
1. Это защита от повторной отправки
2. Удалите существующий файл из output_folder
3. Или запустите тестовую отправку снова

## 🔄 Автоматическое обновление календаря

**Хорошая новость:** Календарь обновляется автоматически!

### Когда происходит обновление:

1. **За 30 дней до нового года** (2 декабря)
   - Автоматически загружается календарь на следующий год
   - Вы получите уведомление в Telegram: "📅 Календарь обновлён!"

2. **При устаревшем календаре**
   - Если текущий год > год в календаре
   - Автоматически загружается актуальный календарь

### Что делать:

**Ничего!** Система сама обновит календарь.

### Если хотите проверить:

```bash
# Смотрим текущий год в календаре
type config\calendar.json | findstr "year"

# Вывод: "year": 2025
```

### Если возникла ошибка обновления:

Вы получите уведомление в Telegram с инструкцией.

Подробнее: [doc/calendar_auto_update.md](doc/calendar_auto_update.md)
