#### ➕ Создать задачу (строки 300-350)
- Создаёт задачу в Windows Task Scheduler
- Вычисляет следующий запуск с учётом календаря
- Использует `runner.py` как точку входа
- Автоматически применяет smart_reschedule

**Логика вычисления следующего запуска:**
```python
from scripts.base_script import BaseScript

base = BaseScript.__new__(BaseScript)
base.config = config

if schedule['type'] == 'daily':
    next_run = base.get_next_run_date('daily', smart_reschedule=True)
else:
    send_day = schedule.get('send_day', 'Thursday')
    next_run = base.get_next_run_date('weekly', send_day=send_day, smart_reschedule=True)

next_run = next_run.replace(hour=hour, minute=minute, second=0, microsecond=0)
```

    capture_output=True,

    messagebox.showinfo("Успех", "Тестовая отправка выполнена успешно!")
