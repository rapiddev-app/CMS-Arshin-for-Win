# ScriptManager: Автоматизация интеграции данных с ФГИС «Аршин»

**ScriptManager** — это десктопное приложение для ОС Windows, созданное для полной автоматизации сбора, фильтрации и отправки метрологических данных в федеральную государственную информационную систему (ФГИС) «Аршин». 

Программа избавляет сотрудников от рутинной ежедневной и еженедельной работы по проверке десятков файлов Excel. Решение самостоятельно собирает нужные реестры поверок (по разным регионам и филиалам), фильтрует их, формирует сводные отчеты, отправляет по электронной почте заинтересованным лицам и уведомляет команду в Telegram.

## 🚀 Ключевые возможности

### 1. Умное расписание и фоновая работа
- **Глубокая интеграция с Windows Task Scheduler:** Программа не требует, чтобы ее окно было постоянно открыто. Приложение управляет Планировщиком задач Windows через COM API-интерфейсы. Фоновые процессы (`runner.py`) выполняются полностью автономно.
- **Интеграция с производственным календарем РФ:** Встроенный парсер проверяет праздничные и выходные дни через API (`isdayoff.ru`). Если день отправки выпадает на выходной, скрипт автоматически принимает решение перенести отправку на ближайший рабочий день.

### 2. Мощный ETL процесс для Excel-данных
- Автоматизированный сбор данных об измерительных приборах из облачных или локальных источников (поддержка параллельного сбора из сводных и разрозненных таблиц Excel).
- Сложная валидация: фильтрация по диапазонам дат (например, за последние 30 дней), исключение пустых серийных номеров («№ в ГРСИ») или некорректных значений.
- Защита от дублей при генерации итоговых файлов реестров (`.xlsx`).

### 3. Информирование и логирование
- **Многоканальные уведомления:** Встроенные драйверы для работы со службами SMTP (Gmail, Яндекс.Почта) для автоматической отправки сгенерированных отчетов-файлов.
- **Telegram бот:** Критические ошибки (отсутствие файла, сбой парсинга) или уведомления об успешной отправке мгновенно отправляются в Telegram (в группу или лично администратору).
- **Централизованное логирование:** Все фоновые действия пишутся в лог-файлы с автоматической ротацией.

### 4. Современный графический интерфейс (GUI)
- Интерфейс разработан с использованием библиотеки **CustomTkinter**, предлагая современный Dark Mode дизайн.
- Удобная панель мониторинга всех задач: пользователь может видеть статусы всех скриптов, даты последних выполнений и время следующего запланированного запуска.
- Защита конфигурации: доступ к Глобальным настройкам (API ключам Telegram, паролям SMTP) защищен мастер-паролем.
- Встроенная вкладка в стиле «Excel-листов» для просмотра лога в режиме реального времени.

## 🛠 Технологический стек

- **Язык программирования:** Python 3.x
- **GUI Фреймворк:** CustomTkinter, Tkinter
- **Работа с данными:** Pandas, OpenPyXL
- **Интеграция с ОС (Windows):** `pywin32` (управление Windows Task Scheduler через `win32com.client`)
- **Сеть и Web:** Requests (API производственного календаря), Telegram Bot API
- **Встроенные библиотеки:** `smtplib`, `email` (генерация MIME-сообщений), `logging`, `json`

## 💡 Решаемая бизнес-проблема

До внедрения ScriptManager сотрудникам метрологического отдела приходилось:
1. Ежедневно мониторить обновления общих таблиц.
2. Вручную копировать строки с датами поверок за последний месяц.
3. Формировать файлы нужного формата.
4. Отправлять письма подрядчикам и в систему ФГИС.

**Результат внедрения:** Процесс полностью переведен в "headless" режим. Программа безошибочно генерирует и отправляет реестры по 6 различным бизнес-юнитам (Воронеж, Липецк, филиалы Квадра, РВК и др.), экономя десятки человеко-часов каждый месяц и исключая фактор человеческой ошибки.
