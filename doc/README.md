# ScriptManager v1.0.0

> Система автоматизации отправки отчётов с интеграцией Windows Task Scheduler и производственного календаря РФ

## 📋 Описание

ScriptManager - это Windows-приложение для автоматической обработки Excel файлов и отправки отчётов по email с использованием планировщика задач Windows.

**Ключевые возможности:**
- ✅ Автоматическая обработка 6 различных типов отчётов
- ✅ Интеграция с Windows Task Scheduler
- ✅ Учёт производственного календаря РФ (праздники, выходные)
- ✅ GUI на базе CustomTkinter
- ✅ Email отправка через Gmail и Yandex SMTP
- ✅ Telegram уведомления
- ✅ Защита настроек паролем
- ✅ Тестовая отправка для проверки

### 🛠️ Вариант 2: Разработка из исходников

**Для разработчиков:**

#### Требования
- Windows 10/11
- Python 3.11.9+
- Excel файлы в OneDrive

#### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/scriptmanager.git
cd scriptmanager
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Настройте конфигурацию:
- Скопируйте `config/email.json.example` → `config/email.json`
- Скопируйте `config/telegram.json.example` → `config/telegram.json`
- Заполните учётные данные

5. Запустите GUI:
```bash
python src/gui.py
```

## 🎯 Скрипты

### CMS 36 Воронеж (daily)
- **Файл:** `cms36_script.py`
- **Расписание:** Ежедневно в 10:00
- **Источник:** `Артур ЦМС.xlsx` → лист `Аршин ЦМС vrn`
- **Фильтр:** Последние 30 дней по `Дата поверки`
- **Выход:** `420 ({count}) {date_range} Воронеж.xlsx`
- **Email:** cms-tab@yandex.ru

### CMS 48 Липецк (daily)
- **Файл:** `cms48_script.py`
- **Расписание:** Ежедневно в 09:00
- **Источник:** Несколько файлов из OneDrive
- **Особенность:** Объединяет данные из 3 файлов
- **Email:** cms-tab@yandex.ru

### Kvadra (weekly)
- **Файл:** `kvadra_script.py`
- **Расписание:** Каждую пятницу в 09:00
- **Фильтр:** По `Дата вноса` (не `Дата поверки`)
- **Email:** cms.lip@yandex.ru (Yandex SMTP)

### Липецктеплосеть (weekly)
- **Файл:** `lts_script.py`
- **Расписание:** Каждый четверг в 09:00
- **Email:** mup-sbyt@mail.ru

### Росводоканал (weekly)
- **Файл:** `rvk_script.py`
- **Расписание:** Каждый четверг в 09:00
- **Особенность:** Многоуровневые заголовки Excel
- **Email:** ev.shlykova@rosvodokanal.ru

### РВК Kvadra (weekly)
- **Файл:** `rvk_kvadra_script.py`
- **Расписание:** Каждый четверг в 09:00
- **Данные:** Kvadra для Росводоканала

## 📅 Производственный календарь РФ

Система автоматически учитывает:
- Выходные дни (суббота, воскресенье)
- Праздничные дни РФ
- Переносы праздников

**Праздники 2025:**
- 1-8 января - Новогодние каникулы
- 23-24 февраля - День защитника Отечества
- 8-10 марта - Международный женский день
- 1-2 мая - Праздник Весны и Труда
- 9 мая - День Победы
- 12-13 июня - День России
- 3-4 ноября - День народного единства

**Умное перепланирование:**
Если дата запуска попадает на выходной/праздник, задача автоматически переносится на следующий рабочий день.

### 🔄 Автоматическое обновление календаря

Система **автоматически** обновляет производственный календарь:

- **За 30 дней до нового года** (2 декабря) загружается календарь на следующий год
- **При устаревшем календаре** (текущий год > год в календаре) загружается актуальный
- **Через API:** https://isdayoff.ru - официальный источник календаря РФ
- **Уведомления в Telegram** при успешном обновлении или ошибке

**Вам ничего делать не нужно!** Календарь обновится автоматически.

Подробнее: [doc/calendar_auto_update.md](doc/calendar_auto_update.md)

## 🔧 Запуск скриптов

### Через GUI
1. Открыть `python src/gui.py`
2. Нажать "➕ Создать задачу" для нужного скрипта
3. Задача автоматически запустится по расписанию

### Вручную через командную строку
```bash
python src/scripts/runner.py cms36
python src/scripts/runner.py cms48
python src/scripts/runner.py kvadra
python src/scripts/runner.py lts
python src/scripts/runner.py rvk
python src/scripts/runner.py rvk_kvadra
```

### Через Windows Task Scheduler
Задачи создаются автоматически через GUI с именами:
- `ScriptManager_cms36`
- `ScriptManager_cms48`
- `ScriptManager_kvadra`
- `ScriptManager_lts`
- `ScriptManager_rvk`
- `ScriptManager_rvk_kvadra`
