# 📦 Руководство по упаковке и распространению ScriptManager

## 🎯 Цель

Создать автономный инсталлятор ScriptManager, который:
- ✅ Не требует установки Python на целевом компьютере
- ✅ Автоматически устанавливается в Program Files
- ✅ Создаёт ярлык на рабочем столе
- ✅ Регистрируется в системе (видим в "Программы и компоненты")
- ✅ Имеет простой деинсталлятор

## 🛠️ Подготовка к сборке

### 1. Установка PyInstaller

```bash
pip install pyinstaller
```

### 2. Проверка зависимостей

Убедитесь, что все зависимости установлены:

```bash
pip install -r requirements.txt
```

## 🚀 Процесс сборки

### Вариант 1: С системным Python

```bash
python build.py
```

**Результат:** Компактный пакет (~50 MB), требует установки Python

### Вариант 2: Автономный (с встроенным Python) ⭐ РЕКОМЕНДУЕТСЯ

#### Шаг 1: Подготовка Embedded Python

```bash
python prepare_embedded_python.py
```

Этот скрипт автоматически:
1. Скачает Python 3.11.9 Embedded (64-bit)
2. Распакует в папку `python_embedded/`
3. Установит pip
4. Установит все зависимости из `requirements.txt`
5. Проверит работоспособность

**Время выполнения:** 3-5 минут
**Размер:** ~120 MB

#### Шаг 2: Сборка release пакета

```bash
python build.py
```

Скрипт автоматически:
1. Очистит предыдущие сборки
2. Соберёт инсталлятор в .exe
3. Включит `python_embedded/` в пакет
4. Создаст папку `release/` с готовым пакетом

**Результат:**
```
release/
├── ScriptManager_Setup.exe  ← Инсталлятор
├── python_embedded/         ← Встроенный Python (если Вариант 2)
│   ├── python.exe
│   ├── python311.dll
│   └── Lib/
│       └── site-packages/   ← Все зависимости уже установлены!
├── src/                     ← Исходники
├── config/                  ← Конфигурация
├── doc/                     ← Документация
├── README.md
├── QUICKSTART.md
├── PROJECT_SUMMARY.md
├── requirements.txt
└── УСТАНОВКА.txt           ← Инструкция для пользователя
```

### Ручная сборка (если нужно)

#### Шаг 1: Сборка инсталлятора

```bash
pyinstaller --onefile --windowed --name=ScriptManager_Setup installer.py
```

#### Шаг 2: Копирование файлов

Скопируйте в папку `release/`:
- `dist/ScriptManager_Setup.exe`
- `src/`
- `config/`
- `doc/`
- `requirements.txt`
- Документацию (README.md, QUICKSTART.md)

## 🖥️ Установка на целевом компьютере

### Требования к системе

**Вариант 2 (Автономный - рекомендуется):**
- **ОС:** Windows 10/11 (64-bit)
- **ОЗУ:** Минимум 2 GB
- **Диск:** Минимум 200 MB свободного места
- **Права:** Администратор для установки
- **Python:** ❌ НЕ требуется! (всё встроено)

**Вариант 1 (С системным Python):**
- **ОС:** Windows 10/11 (64-bit)
- **ОЗУ:** Минимум 2 GB
- **Диск:** Минимум 100 MB свободного места
- **Права:** Администратор для установки
- **Python:** ✅ Требуется Python 3.11.9+

### Процесс установки

1. **Запуск инсталлятора:**
   ```
   Правый клик на ScriptManager_Setup.exe
   → Запуск от имени администратора
   ```

2. **Выбор пути установки:**
   - По умолчанию: `C:\Program Files\ScriptManager`
   - Можно изменить на любой другой

3. **Опции установки:**
   - ☑ Создать ярлык на рабочем столе (рекомендуется)
   - ☐ Добавить в автозагрузку Windows (опционально)

4. **Нажать "Установить"**

5. **Дождаться завершения:**
   ```
   Создание директории установки...
   Копирование файлов...
   Создание ярлыка на рабочем столе...
   Регистрация в системе...
   Создание деинсталлятора...
   ```

6. **Готово!**
   - Ярлык на рабочем столе
   - Запись в "Программы и компоненты"

## 🗑️ Удаление приложения

### Способ 1: Через Windows

```
Параметры Windows
→ Приложения
→ ScriptManager
→ Удалить
```

### Способ 2: Через деинсталлятор

```
C:\Program Files\ScriptManager\uninstall.bat
→ Запустить от имени администратора
```

### Что удаляется:

- ✅ Все файлы из `C:\Program Files\ScriptManager`
- ✅ Ярлык с рабочего стола
- ✅ Ярлык из автозагрузки (если был)
- ✅ Запись из реестра Windows
- ✅ Задачи из Task Scheduler (вручную)

### Что НЕ удаляется:

- ❌ Конфигурационные файлы с паролями (сохранены в `config/`)
- ❌ Логи (сохранены в `logs/`)
- ❌ Excel файлы пользователя

**Для полной очистки:** Вручную удалите папку после деинсталляции.

## 📊 Сравнение вариантов упаковки

### Вариант 1: С системным Python

**Плюсы:**
- ✅ Компактный размер (~50 MB)
- ✅ Легко обновлять Python отдельно
- ✅ Быстрая сборка

**Минусы:**
- ❌ Требует установки Python на целевом компьютере
- ❌ Требует ручной установки зависимостей
- ❌ Возможны конфликты версий Python

**Когда использовать:**
- Python уже установлен на всех целевых компьютерах
- Есть централизованное управление ПО
- Минимальный размер дистрибутива критичен

---

### Вариант 2: Автономный (Embedded Python) ⭐

**Плюсы:**
- ✅ Не требует установки Python
- ✅ Полностью автономный
- ✅ Все зависимости уже встроены
- ✅ Один клик - и работает
- ✅ Нет конфликтов версий

**Минусы:**
- ❌ Больше размер (~150 MB)
- ❌ Дольше сборка (3-5 минут)

**Когда использовать:**
- Python НЕ установлен на целевых компьютерах
- Нужна простая установка "из коробки"
- Пользователи не технические специалисты
- **РЕКОМЕНДУЕТСЯ для большинства случаев**

## ❓ Частые вопросы

### Q: Можно ли использовать без прав администратора?

A: Нет, для установки нужны права администратора. Но после установки можно запускать без них.

### Q: Работает ли на Windows 7?

A: Нет, требуется Windows 10/11. Python 3.11 не поддерживает Windows 7.

### Q: Можно ли установить несколько копий?

A: Да, просто укажите разные пути установки.

### Q: Как сделать инсталлятор меньше?

A: Используйте `pyinstaller --onefile --upx-dir=upx` (требует UPX compressor)

### Q: Можно ли добавить иконку?

A: Да:
```bash
pyinstaller --onefile --windowed --icon=icon.ico installer.py
```
